# 🚀 دليل النشر على Netlify

## كيفية نشر موقعك على Netlify باستخدام Youssef AI

### الخطوة 1: الحصول على Netlify Personal Access Token

1. **إنشاء حساب Netlify** (إذا لم يكن لديك حساب):
   - اذهب إلى [https://app.netlify.com/signup](https://app.netlify.com/signup)
   - سجّل حساب جديد (يمكنك استخدام GitHub أو Email)

2. **الحصول على Personal Access Token**:
   - بعد تسجيل الدخول، اذهب إلى: [https://app.netlify.com/user/applications#personal-access-tokens](https://app.netlify.com/user/applications#personal-access-tokens)
   - أو من القائمة: `User Settings` → `Applications` → `Personal Access Tokens`
   - اضغط على **"New access token"**
   - أدخل اسماً للـ Token (مثل: "Youssef AI")
   - انسخ الـ Token **فوراً** (لن تتمكن من رؤيته مرة أخرى!)

3. **إضافة الـ Token في Youssef AI**:
   - افتح Youssef AI
   - اضغط على زر **API Keys** (الأيقونة البنفسجية في الأعلى)
   - أضف الـ Netlify Token في الحقل المخصص
   - احفظ الإعدادات

### الخطوة 2: نشر موقعك

1. **توليد الكود**:
   - استخدم Youssef AI لإنشاء موقعك
   - انتظر حتى يكتمل التوليد

2. **النشر على Netlify**:
   - اضغط على زر **"Publish to Netlify"** (الزر الأخضر)
   - انتظر قليلاً... سترى السجلات في الزاوية السفلية اليمنى

3. **الحصول على الرابط**:
   - بعد اكتمال النشر، ستظهر نافذة النجاح
   - انسخ رابط موقعك
   - شارك الرابط على وسائل التواصل الاجتماعي!

## 📋 مراحل النشر

عند الضغط على زر النشر، ستمر العملية بالمراحل التالية:

1. **🚀 Starting deployment process**
   - بدء عملية النشر

2. **📦 Fetching files from sandbox**
   - جلب الملفات من Sandbox

3. **✅ Found X files to deploy**
   - عرض عدد الملفات المراد نشرها

4. **🌐 Creating site**
   - إنشاء موقع جديد على Netlify

5. **⬆️ Uploading files to Netlify**
   - رفع الملفات إلى Netlify

6. **✅ Files uploaded successfully**
   - اكتمال رفع الملفات

7. **🔄 Processing deployment**
   - Netlify يقوم بمعالجة الموقع

8. **🎉 Deployment completed!**
   - اكتمل النشر بنجاح!

## ❓ حل المشاكل الشائعة

### المشكلة: "Netlify token not provided"
**الحل**: تأكد من إضافة الـ Token في إعدادات API Keys

### المشكلة: "Failed to authenticate with Netlify"
**الحل**: 
- تأكد من صحة الـ Token
- تأكد أنك نسخت الـ Token كاملاً
- جرب إنشاء Token جديد

### المشكلة: "Failed to create Netlify site"
**الحل**: 
- قد يكون اسم الموقع محجوز
- جرب مرة أخرى (سيتم إنشاء اسم جديد تلقائياً)

### المشكلة: "No files found in sandbox"
**الحل**: 
- تأكد من توليد الكود أولاً
- تأكد من وجود Sandbox نشط

## 🎯 نصائح

1. **احتفظ بالـ Token في مكان آمن** - لا تشاركه مع أحد
2. **اسم الموقع** - يتم توليده تلقائياً بصيغة `youssef-ai-XXXXX`
3. **إعادة النشر** - يمكنك النشر أكثر من مرة، وسيتم إنشاء موقع جديد في كل مرة
4. **الرابط المباشر** - الرابط يعمل فوراً ولا يتغير
5. **المشاركة** - استخدم أزرار المشاركة في نافذة النجاح للمشاركة على وسائل التواصل
