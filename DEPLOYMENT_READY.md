## 🎯 كيف تستخدم النظام

### للنشر على Netlify:
```bash
1. احصل على Token من Netlify
   https://app.netlify.com/user/applications#personal-access-tokens

2. أضف Token في API Keys (الزر البنفسجي)

3. ولّد موقعك

4. اضغط "Publish to Netlify"

5. انتظر 10 ثواني → احصل على الرابط!

6. انتظر 1-2 دقيقة → الموقع جاهز!
```

### لإضافة الصور:
```bash
فقط اطلب:
"اعمل موقع مطعم مع صور طعام جميلة"
"موقع سفر بصور مناظر طبيعية"
"متجر إلكتروني بصور منتجات"

والـ AI سيضيف الصور تلقائياً! 🎨
```

### للتصميم المتجاوب:
```bash
✨ لا تحتاج لفعل شيء!
✨ التصميم متجاوب تلقائياً
✨ يعمل على جميع الأجهزة
```
