🚀 دليل سريع: النشر على Netlify في 3 خطوات
═══════════════════════════════════════════════════════════

الخطوة 1️⃣: احصل على Token من Netlify
   👉 اذهب إلى: https://app.netlify.com/user/applications#personal-access-tokens
   👉 اضغط "New access token"
   👉 انسخ الـ Token فوراً!

الخطوة 2️⃣: أضف الـ Token في Youssef AI
   👉 اضغط على زر API Keys (الأيقونة البنفسجية 🔷)
   👉 الصق الـ Token في حقل "Netlify Token"
   👉 احفظ

الخطوة 3️⃣: انشر موقعك!
   👉 اضغط على زر "Publish to Netlify" (الزر الأخضر ▲)
   👉 انتظر قليلاً (30-60 ثانية)
   👉 احصل على رابط موقعك! 🎉

═══════════════════════════════════════════════════════════
📝 ملاحظات مهمة:
═══════════════════════════════════════════════════════════

✅ الـ Token يُستخدم مرة واحدة للإعداد فقط
✅ موقعك سيكون متاح عالمياً على الإنترنت
✅ الرابط يعمل فوراً ولا يتغير
✅ يمكنك المشاركة على وسائل التواصل
✅ SSL مجاني (HTTPS) تلقائياً

═══════════════════════════════════════════════════════════
🆘 في حال واجهت مشكلة؟
═══════════════════════════════════════════════════════════

❌ "Token not provided" → أضف الـ Token في API Keys
❌ "Authentication failed" → تحقق من صحة الـ Token
❌ "No files found" → ولّد الكود أولاً
❌ "Site creation failed" → جرب مرة أخرى
